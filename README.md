Tic Tac Toe AI Game

A desktop-based Tic Tac Toe game where a human player competes against an AI opponent powered by the Minimax algorithm with Alpha-Beta Pruning.

---

Project Overview
This project demonstrates fundamental Artificial Intelligence concepts through a classic Tic Tac Toe game, where the AI always plays optimally and cannot be defeated.

---

AI Techniques Used
- Minimax Algorithm for optimal move selection
- Alpha-Beta Pruning to reduce the search space and improve efficiency
- Game Tree Evaluation to determine winning, losing, and draw states

---

Technologies Used
- Python
- Tkinter (GUI Development)
- Artificial Intelligence Algorithms

---

Game Features
- Human vs AI gameplay
- AI makes optimal decisions
- Clear visual feedback for turns and game status
- Detection of win, loss, or tie
- Option to start a new game

---

User Interface
- Clean and modern desktop UI
- Interactive buttons for gameplay
- Color-coded player symbols
- Real-time game status updates
